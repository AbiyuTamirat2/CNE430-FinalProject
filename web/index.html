Technical Overview

This project leverages Amazon Web Services (AWS) to host the online resume hub with a focus on scalability, security, and reliability. The setup utilizes:

  • Amazon S3 for static website hosting
  • Route 53 for DNS management
  • CloudFront for content delivery and caching
  • AWS Certificate Manager (ACM) for SSL/TLS encryption

This combination ensures resumes are securely accessible and optimized for fast loading across the globe, demonstrating best practices for hosting static websites on AWS.
